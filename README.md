# Шаг Вперёд

![Тут должно быть изображение =)](/photos/readme.png)
​
 **Шаг Вперёд** — это информационная система для компании, занимающейся продажей обуви. Оно предоставляет удобные средства для управления товарами и заказами, а также поддерживает работу разных типов пользователей: гостя, авторизованного клиента, менеджера и администратора.

Система включает собственную базу данных, пользовательский интерфейс, механизмы авторизации, обработки товаров и заказов, а также инструменты для просмотра, фильтрации, поиска и редактирования данных.

## Основные функции

- просмотр списка товаров;
- сортировка, фильтрация, поиск (для менеджера и администратора);
- авторизация и управление аккаунтами;
- добавление/редактирование/удаление товаров (админ);
- работу с заказами (менеджер, админ);
- хранение изображений товаров;
- разработка интерфейса по макетам.

## Запуск приложения

 1. **Установите зависимости:**

     ```bash
     pip install -r requirements.txt
     ```

 2. **Настройте базу данных:**

     - Убедитесь, что у вас запущен PostgreSQL.
     - Создайте базу данных и пользователя в соответствии с настройками в `main.py`.

 3. **Запустите приложение:**

     ```bash
     python main.py
     ```

## Структура проекта

- `main.py`: Основная точка входа в приложение. Отвечает за инициализацию базы данных, настройку пользовательского интерфейса и запуск приложения.

- `database/`: Содержит логику для взаимодействия с базой данных PostgreSQL.

- `presentation/`: Содержит все компоненты пользовательского интерфейса, разделенные на различные модули:

  - `Login_UI/`: Пользовательский интерфейс для входа в систему.
    - `Navigation/`: Управление навигацией между различными окнами.
    - `Edit_order_UI/`: Пользовательский интерфейс для редактирования заказов.
    - `Order_list_UI/`: Пользовательский интерфейс для отображения списка заказов.
    - `Product_edit_UI/`: Пользовательский интерфейс для редактирования товаров.
    - `Product_list_UI/`: Пользовательский интерфейс для отображения списка товаров.
- `res/`: Содержит ресурсы, такие как иконки и изображения.
